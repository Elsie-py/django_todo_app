#Django To-Do List Application

## Project Overview
A full stack Django To Do list application built using
- Django
- Django Rest Framework
- SQLite(development)
- HTML Templates

The project demonstrates CRUD functionality, REST APIs, and Django core concepts

## Features
- Create, update, delete tasks
- Mark tasks as complete with default not complete
- Web interface using Django templates
- REST API endpoints for tasks
- Admin panel for data management

## Technologies Used
- Python
- Django
- Django Framework
- HTML
- Git & Github
